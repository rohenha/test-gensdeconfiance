# Test Gens de confiance

## Stack 
- Gatsby
- React
- SASS

## Lancement du projet
- Installer les dépendances (yarn install)
- Lancer le server gatsby dans le terminal (gatsby develop)
- Ouvrir la page [http://localhost:8000](http://localhost:8000)
